1. 主要观点与论点

A. 写作语法核心：表达要“指代清楚、结构完整”

英语里一句话必须有明确的主语+谓语；如果出现新动作/新句子，需要明确“谁做的”。

两句完整句不能只用逗号硬连（中文逗号思维不适用），否则会造成：

结构错误（comma splice）

指代不明（后半句“谁做的”不清楚）

B. 用词与语气：建议/倡议要用“should”，别用“can”太客气

当写环保倡议时，语气更像“应该做”，因此倾向用：

We should… / We must…

We can… 更像“我们能够/可以（有能力）”，语气偏客气或偏能力描述，不够“倡议感”。

C. 题目与内容要一致：避免“跑题”

如果标题是 The Environment（环境），正文却主要写 Protecting the environment（保护环境），会被认为“相关但偏题”。

更匹配的标题如：

Ways of Protecting the Environment

How to Protect the Environment

2. 重要事实与规则（含例句/纠错点）

A. “to” 的核心语义：动作/结果真正落到对象身上

老师用“买书是否到手”来解释：

我说要给你买（还没给到）：像“画饼”，结果未发生

我已经给到你手上：动作结果实现 → 用 to 的“给到/指向”

这段的本质：强调英语里介词/结构常带“结果是否落实”的语感。

B. “now” 是否多余：看语法与语义是否重复

例：…is getting worse day by day now

day by day 已经表达“逐日变差”

is getting… 也自带“正在变化”的语感

所以 now 可能显得重复（可删或改写更自然）

C. 动词变“主语”要名词化：用动名词（V-ing）

“保护环境变得越来越重要”正确结构：

✅ Protecting the environment becomes more and more important.

关键点：

Protecting the environment 作为整体是一件事 → 视为单数

所以谓语用 becomes（三单）

D. “越来越……”固定结构

✅ more and more + 形容词

more and more important = 越来越重要

区分：

more important = “更重要（与别的比较）”

more and more important = “重要性逐渐上升”

E. 逗号不能当连接词：两句要用连接词或断句

典型问题：前面一句完整，逗号后又来一句完整，但没连接词、还可能没主语。

正确做法：

用 and/but/so/because… 等连接词

或者直接用句号分成两句

后半句缺主语时，可补 it 指代前面整件事（“保护环境这件事”）

F. 从句里要用“动词”，不能把动词穿成“名词马甲”

纠错点：

❌ when we going out

✅ when we go out

原因：when 引导的是从句，需要谓语动词（go），而不是动名词（going）当名词用。

G. 代词指代要匹配数量

文中有两个动作/两个点时，用 it 会不合适：

如果前面提了不止一个事项 → 用 them / these / this 更合适

例：
If everyone can do them/these, the environment will be better.

优点：

有层次：First / Second

有举例：for example

逻辑清晰，整体不错

需要提升：

第二点缺少具体例子（第一点举例更充分）

可以适当增加细节与字数，让文章更“饱满”、更有说服力

1. 主要观点和论点

A. talk 的三种常见搭配（考试常考固定搭配）

talk with sb：与某人交谈（强调“和……一起聊”，互动感强）

talk to sb：对某人说/与某人交谈（强调“对着……说”，有时更像单向）

talk about sth/sb：谈论某事/某人（强调“内容主题”）

老师的意思：
talk 的重点主要在搭配；talk to 在很多语境里接近 “tell”，但 tell 更像“告诉/通知（信息）”，语感更直接。

B. speak vs say：一个偏“方式/语言能力”，一个偏“说的内容”

speak：强调“说话的方式/使用某种语言”

speak English / speak Chinese：表示“会用这种语言说话”，不是把“English”这个词念出来

speak up：大声点说/说清楚（方式）

say：强调“说的内容是什么”

say “It will rain.”（内容是“将要下雨”）

你要“把某个词说出来”，用 say 更贴近：say “English”.

C. 这四个动词的本质差别（老师总结）

say：说“内容”

speak：说“语言/方式”

tell：告诉“某人”（对象明确，常带信息传递/指令）

talk：交谈（更像“聊天/讨论”，重搭配）

2. 重要事实和数据（规则、变形、结构、易错点）

A. 过去式 & 过去分词（重点是“形式 + 用法”）

老师强调：考试不光拼写，还可能考听力读音。

1. 四个动词的变形（老师点名要记）

tell – told – told

talk – talked – talked（规则变化）

speak – spoke – spoken（不规则）

say – said – said（said 读音老师提醒：听力会出现）

2. 过去式的用法（Past Simple）

表示“过去发生的动作/状态”

I ate rice yesterday.（动词穿“过去式马甲”）

3. 过去分词的三大用法（老师要求你能背出来）

现在完成时：have/has + 过去分词

I have already done my homework.

被动语态：be + 过去分词

The apple is eaten (by me).（苹果被吃）

作形容词（过去分词当形容词修饰名词）

an eaten apple = “被吃过的苹果”

B. 被动语态结构（老师要求能秒答）

be + 过去分词 (+ by + 动作执行者)

be 的形态要熟：am / is / are / was / were / been / being

老师特别点：have been（现完里的 been）、is being（进行时被动）也要认识

例句（老师带你推的逻辑）：

主语换成“苹果”，因为苹果不能“吃别人”，只能“被吃”

✅ The apple is eaten by me.

✅ I eat the apple.（主动）

C. 常见 ABB / AAA 变化（老师列了一串，意思是：见到要反应快）

ABB 型：say–said–said（原形≠过去式=过去分词）

AAA 型（三态同形）示例：

cut–cut–cut

put–put–put

hit–hit–hit

cost–cost–cost

spread–spread–spread

read–read–read（拼写一样，但读音会变，老师提醒别按拼写乱读）

D. ask 的两种常考结构（老师后面讲练习题时提到）

ask sb to do sth：请求/要求某人做某事

ask sb for sth：向某人要某物

（老师的口诀）：很多时候 to do ≈ for + 名词 的替换思路能帮助理解，但题里要按固定搭配选。

E. 练习题中出现的关键纠错点（非常像考试点）

have done：do 的过去分词是 done（不是 do / did）

be controlled：出现 “be + 过去分词” 多半是被动，不要误读成“正在控制”

“The house is controlled by a central system.” = 房子被中央系统控制

speaker vs speech（老师强调词性）

speaker：人/说话者（名词）

speech：演讲（名词）

speak：演讲/说（动词）
老师指出你把 speaker 当成“语音助手”那种理解会跑偏；语境里更可能是“演讲”。

proper preparation：合适/充分的准备

experiment：实验（不要和 experience 混）

1. 主要观点和论点

A. 长句翻译方法：先找“介词/连词”，从后往前翻

英语长句变长的原因通常是介词短语/从句多。

翻译时先定位：

介词（in / for / to / without / with…）

连词（because / when / if / that…）

把这些当成“附加成分”先拆出来，再从后往前组合，更顺。

B. 介词后面只能接“名词性成分”

介词 + 名词 / 代词 / 动名词(V-ing) / 名词短语

不能直接接动词原形（除了 to do 这个特殊结构里的 to，老师强调：它不是普通介词用法）

C. 固定搭配比“感觉”更可靠

例如：

invite sb to do sth ✅（邀请=让对方“将要去做”）

ask sb to do sth ✅（请求/要求某人去做）

see sb do / doing ✅（两种结构含义不同，见下）

2. 重要事实和规则（例句+易错点）

A. 关键词释义（proper / essential / experiment / experience）

proper：合适的、恰当的

essential：极其重要的（≈ very important）

experiment：实验（偏科学/化学/物理）

experience：经历 / 经验（同一个词两层意思）

例句翻译（老师示范拆句）

Proper preparation is essential for a successful experiment in science class.

拆法：

in science class：在科学课上（介词短语）

for a successful experiment：对于一个成功的实验来说（for = “对……而言/为了……”）

组合成顺的中文：

✅ 在科学课上，对于一个成功的实验来说，充分/合适的准备是非常重要的。
（你也可以把“proper”译成“充分的/恰当的”都可以）

B. 为什么 preparation 是名词？
为什么不是 prepare？

prepare 是动词：准备

preparation 是名词：准备（这件事/准备工作）

形容词 + 名词结构：proper preparation（恰当的准备）

C. “动词-ing”为什么能当主语/名词性？

V-ing 可以“名词化”，当作一件事：

Giving a speech is…（做演讲这件事……）

主语往往是一个整体（动名词短语），谓语按单数处理。

D. 介词后面为什么要用名词性？（management 这个点）

老师讲了一个典型题思路：

into 是介词 → 后面要 名词性

manage（动词）要变名词：

management：管理（名词）

manager：经理（名词，人）

例句意思（按你段落）：

We need to get women into management.

✅ 我们需要让女性进入管理层/从事管理工作。

关键记法：介词 + 名词性（名词/动名词/代词），不要加 -ed 乱造形式。

E. see sb do / doing 的区别（老师明确讲了）

see sb do sth：看见某人做了（全过程），不强调“正在”

see sb doing sth：看见某人正在做（进行中）

例：

I saw him speak English fluently.（看到他完整展示“会说”）

I saw him speaking English.（看到他当时正在说）

你那句整体意思：

“看见如此小的男孩能流利说外语令人惊讶”
常用：see sb do（不强调正在说的瞬间）

F. invite / ask 的固定搭配（不能用 doing）

invite sb to do sth ✅
不能 invite sb doing sth（邀请=未来动作，“他都没来怎么 doing”）

ask sb to do sth ✅（请求/要求某人做）

ask sb for sth ✅（向某人要某物）

G. without 后面跟什么？（老师当场考点）

without 是介词

介词后接：名词 / 代词 / 动名词(V-ing)

without help（没有帮助）

H. 称呼：Miss / Mrs / Ms / Mr（老师纠错点）

Mr.：男性（不管结婚没结婚）

Mrs.：已婚女性
